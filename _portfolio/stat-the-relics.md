StatTheRelics displays live and historical stat counters for relics in *Slay the Spire 2*. It patches relic tooltips to append useful run-specific data, then persists those counters into run saves and run history.

The mod tracks relic-specific metrics, restores saved statistics when viewing past runs, and stores versioned sidecar data so incompatible statistics are not shown after an update.

The mod is available on the [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3750161122), and the source code is available on [GitHub](https://github.com/Lei-Tin/StatTheRelics).
